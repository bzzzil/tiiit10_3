# Email Finder in C++

Эта программа на C++ выполняет поиск email-адресов в тексте с использованием регулярных выражений.

## Описание программы

Программа:

1. Принимает строку текста с несколькими email-адресами.
2. Использует регулярное выражение для поиска всех email-адресов в тексте.
3. Выводит найденные email-адреса построчно.
